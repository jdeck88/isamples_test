About the iSamples Project

iSamples is designed to create an infrastructure for registering and managing material samples with persistent identifiers (PIDs) and standard metadata, enabling better data reuse and scientific discoveries.

Project Deliverables

iSamples Central (iSC)

Design and development of the iSamples Central architecture and API, supporting use cases, integrations, and a production deployment hosted by CyVerse.

iSamples in a Box (iSB)

Deployable system for local sample management, including API development, user interfaces, and support for standard identifiers like IGSN and ARKs.

Community & Standards Integration

Mapping core metadata to MIxS, Darwin Core, IGSN, and DataCite standards to ensure broad interoperability.

Use Case Testing

Deployment and testing with key partners: Open Context, SESAR, GEOME, and Smithsonian Institution/EMU.

Community Engagement

Hackathons, workshops, and outreach programs for developers and researchers, fostering international collaboration.

Usage Metrics & Impact

Tracking adoption and impact through sample registrations, system queries, metadata completeness improvements, and contributions to iSamples code.

Privacy protections ensure ethical and secure data collection in compliance with industry standards.
